# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-03-02

### Added
- Initial release of Yamori
- Support for TOML and YAML test configuration files
- Terminal UI (TUI) mode with interactive test result visualization
- CLI mode for CI/CD integration
- Test history tracking
- Support for global and per-test build configurations
- Color-coded test results with diff view
- Timeout support for test commands
- Pre-build command execution
- Release/debug mode toggle
